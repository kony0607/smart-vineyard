01 システムアーキテクチャ概要

本システムは、ブドウ園における雨よけビニールの開閉を、気象データおよび現地のセンサ情報に基づいて完全自動化することを目的としています。

エッジ層

Raspberry Piによるモータ制御、雨滴センサ・温湿度センサのデータサンプリング。

通信層

MQTTまたはHTTPを用いたFirebase Realtime Databaseとのリアルタイム同期。

アプリケーション層

React/Next.jsによる管理画面。LINE Messaging APIを用いたアラート通知。

02 機械・構造設計 (Fusion 360)

R800 円弧ラック機構

設計手法Python API (Parametric)
モジュール2.0 (Customized)
曲率半径800 mm
材質ASA (高耐候性フィラメント)

Pythonによる自動生成モデル

2面固定マウント

15x15アルミフレームと既存の木製支柱を接合するための専用ブラケット。Fusion 360の「アセンブリ」機能により、実機とのクリアランスを0.2mm単位で調整。

  • M4 ボルト/ナット固定対応
  • インフィル密度 60% (強度重視)

03 電子回路・電力供給

Controller Unit

Main MCU Raspberry Pi 4 Model B
Motor Driver TB6600 (High Torque Type)
Power Supply AC 100V to DC 24V 10A

Sensor Network

雨滴センサ: 静電容量方式により降雨の瞬瞬を検知。
リミットスイッチ: 原点復帰および脱調検知用。
