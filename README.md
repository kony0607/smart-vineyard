# Smart Vineyard Project

岩手県大迫町のワイン産業を守るためのスマート雨よけシステムの開発記録。

## プロジェクト概要

近年の気候変動による不定期な豪雨から、ブドウを病害から守るために**自動開閉式の雨よけシステム**を開発しました。

Raspberry Pi、3Dプリント、機械加工を組み合わせた、安価で現場に寄り添ったソリューションです。

## 技術スタック

### ハードウェア
- **機械設計**: Fusion 360（API / Pythonパラメトリック設計）
- **制御基板**: Raspberry Pi 4
- **駆動**: ステッピングモータ + ラック&ピニオン機構
- **3D出力**: ASA樹脂（耐候性）

### ソフトウェア
- **フロントエンド**: React + TypeScript + Tailwind CSS + Vite
- **バックエンド**: Firebase Realtime DB
- **組み込みソフト**: Python on Raspberry Pi
- **通知**: LINE Messaging API

## 主な特徴

✅ **高精度ラック設計**: Fusion 360 API で R800 曲線に完璧にフィットする 12.38mm ピッチを自動生成  
✅ **耐久性**: ベアリング・シャフト取付部は機械加工で精密加工  
✅ **農家向けUI**: 説明書がいらないほど直感的な操作画面  
✅ **天気連携**: リアルタイム気象情報から自動で開閉判断  
✅ **年間 50 時間の労働削減**: 手動作業から自動化へ  

## ポートフォリオを見る

ブラウザで `web.html` を開いてください。

```bash
# ローカルサーバーで実行（オプション）
python -m http.server 8000
# または
npx http-server
```

その後 `http://localhost:8000` にアクセス。
